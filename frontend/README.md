# ProxyForge Frontend

This is the Next.js 15 frontend for the ProxyForge platform.

## 🚀 Features
- **Dashboard:** Real-time proxy validation and management.
- **API Engine:** API key management and usage tracking.
- **Network:** Global mesh visualization and node provisioning.
- **Docs:** Integrated documentation center.

## 🛠 Tech Stack
- **Next.js 15** (App Router)
- **Framer Motion** (Animations)
- **Tailwind CSS** (Styling)
- **Lucide Icons** (Visuals)

## 📦 Deployment
The frontend is designed to be built via Docker using the optimized multi-stage `Dockerfile` in this directory.

For full project documentation and setup instructions, please see the [Root README](../README.md).
